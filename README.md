## Reproducibility

For the python version:

```
python3.12 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install numpy pandas matplotlib statsmodels jupyter
```

Then, to build the Python version of the slides

```
make py.pdf
```
